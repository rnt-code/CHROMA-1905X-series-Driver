Format string para que el string resultante (comando) tenga como separador decimal el punto (.):
%#_g (original a reemplazar)
%.;%#_g (actual)

En PCs donde el SO tenga la coma (,) como separador decimal, se deberá agregar %. al format string para que el número cambie la coma (,) por punto (.). 

Ejemplo de SCPI command:
Limit:ARC:LEVel %.;%f
